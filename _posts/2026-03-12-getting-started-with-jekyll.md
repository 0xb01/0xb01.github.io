## Why Jekyll?

[Jekyll](https://jekyllrb.com/) is a simple, blog-aware, static site generator perfect for personal, project, or organization sites. Think of it like a file-based CMS, without all the complexity. Jekyll takes your content, renders Markdown or Liquid templates, and spits out a complete, static website ready to be served by Apache, Nginx or another web server.

## Key Features

- **Simple**: No more databases, comment moderation, or pesky updates to install—just your content.
- **Static**: Markdown, Liquid, HTML & CSS go in. Static sites come out ready for hosting.
- **Blog-aware**: Permalinks, categories, pages, posts, and custom layouts are all first-class citizens.

## Getting Started

To get started with Jekyll, you'll need to:

1. Install Ruby (version 2.5.0 or higher)
2. Install Bundler: `gem install bundler`
3. Clone your repository
4. Run `bundle install` to install dependencies
5. Run `bundle exec jekyll serve` to start the development server

That's it! Your site will be available at `http://localhost:4000`.
